## Initiation message

```
+----------------+--------------+
| *offset (len)* | *value*      |
+----------------+--------------+
| 0 (2)          | Version      |
| 2 (8)          | Timestamp    |
+----------------+--------------+
```


## File List Transfer

```
+---------------------------+----------------+
| *offset (len)*            | *value*        |
+---------------------------+----------------+
| 0 (4)                     | File Count     |
| 4 (n * (128 + 8 + 1 + 1)) | File name list |
+---------------------------+----------------+
```

[See struct s_sync_location](../sync.h)
